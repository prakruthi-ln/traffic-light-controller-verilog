# traffic-light-controller-verilog

RTL Design of Traffic Light Controller using Verilog HDL

\#Traffic Light Controller using Verilog HDL 🚦



\## 📌 Overview

Finite State Machine (FSM) based Traffic Light Controller designed using Verilog HDL.



\## 🧠 Design Concept

\- Synchronous FSM

\- 4 States

\- Timer-based transitions

\- Reset support



\## 📂 Project Structure

\- src/ → RTL code

\- tb/ → Testbench

\- docs/ → FSM diagram



\## 🧪 Simulation

Verified using Vivado Simulator.



\## 🎯 Skills Demonstrated

\- RTL Design

\- FSM Design

\- Verilog Coding
